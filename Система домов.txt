команды системы домов (выполняются ТОЛЬКО RCON-администратором!!!)

рекомендованные команды:

/removeallhouses - удаляем все дома сервера
/changeallprices - На всех домах установить стоимость дома
/sellallhouses - продать все дома
/createhouse - создать дом
/removehouse - удалить дом по ID
/changeprice - установить стоимость дома по ID
/sellhouse - продать дом по ID
/gotohouse - телепортироваться к дому по ID
/passhouse - просмотреть пароль дома
/relhouses - перезагрузить систему домов
/lchouse - заблокировать дом по ID
/ghcmds - список доступных команд

все команды:

/removeallhouses - удаляем все дома сервера
/changeallprices - На всех домах установить стоимость дома
/removeallhcars - удалить автомобиль возле дома у всех
/sellallhouses - продать все дома
/createhouse - создать дом
/removehouse - удалить дом по ID
/changeprice - установить стоимость дома по ID
/changespawn - изменить spawn положение дома
/removehcar - удалить автомобиль возле дома по ID
/sellhouse - продать дом по ID
/gotohouse - телепортироваться к дому по ID
/relhouses - перезагрузить систему домов
/lchouse - заблокировать дом по ID
/addhcar - добавить положение автомобиля по ID
/changehcar - Изменить автомобиль для дома по ID
/ghcmds - список доступных команд
